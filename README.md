Weather Forecast App
React weather app that shows real-time weather for any city using OpenWeatherMap API.

Features
Real-time weather data
City search
Mobile responsive
Modern UI

Setup
Clone repo
npm install
Get API key from OpenWeatherMap
Add key to App.js
npm start

API
OpenWeatherMap Current Weather API

Tech
React 18
CSS3
OpenWeatherMap API
